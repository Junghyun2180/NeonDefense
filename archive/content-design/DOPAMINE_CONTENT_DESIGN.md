# NeonDefense — 도파민 / 콘텐츠 / 리텐션 설계

> 엔지니어링은 안정됐지만 "다시 켤 이유"가 부족하다는 판단에서 시작.
> 각 요소를 도파민 트리거 관점으로 분해하고, 임팩트/비용 기준 Tier로 분류.

## 설계 원칙

1. **Variable Reward** (Skinner) — 예측 불가 보상이 도파민 최대
2. **Near-miss Effect** — "이번엔 진짜 될 뻔" 이 재시도 욕구 핵심
3. **Progress Bar Hook** — 다음 마일스톤까지 남은 거리 항상 표시
4. **Loss Aversion** — 잃을 것이 있어야 조심함 (= 몰입)
5. **Surprise & Delight** — 10분 안에 최소 1번 예상 외 연출

## 우선순위 — Tier 분류

### 🥇 Tier S — 즉시 효과, 저비용 (4시간 이내)
각 1-2시간. 오늘 세션 내 완료 목표.

#### S1. 레어 Prism 가챠 + Pity 시스템
- 뽑기 시 **0.5% 확률**로 "Prism" 등급 T1 출현
  - Prism: 모든 스탯 ×1.30, 무지개 이펙트, Rainbow 속성 (모든 속성으로 조합 가능)
  - 뽑히는 순간 화면 전체 플래시 + 사운드
- **Pity 시스템**: 10연뽑 내에 T2+ 하나 보장 (실제 10번 모두 T1이면 마지막 T1 하나를 강제 T2로 승급)
- **Prism 천장**: 200연뽑 누적 후 Prism 확정 (localStorage 카운터)
- 영향: 가챠 의존 게임에서 필수. 수익화 시 "광고 보면 Prism 확률 ×2" 훅 연결

#### S2. 킬 콤보 카운터
- 2초 내 연속 킬 = 콤보 카운트. 2초 이내 추가 킬 없으면 리셋
- 화면 중앙에 큰 숫자 `12 KILLS!` 펄스 애니메이션
- 마일스톤 (10/25/50/100): 화면 플래시 + 크리스탈 보너스
- 최대 콤보는 게임 결과에 기록, 리더보드/도전과제 연동

#### S3. Near-miss 연출
- **lives ≤ 5**: 화면 테두리 붉은 vignette + `⚠️` 아이콘 상단 중앙 깜빡임
- **lives ≤ 2**: vignette 더 진하게 + BGM 피치 -10% (긴박)
- **lives = 0 직전**: 슬로우 모션 효과 (gameSpeed 일시 0.5x)
- 웨이브 마지막 적에 "BOSS" 스타일 강조 테두리

### 🥈 Tier A — 리텐션 축 (1-2일)
다음 세션 우선순위.

#### A1. 스테이지 별점 (1~3★)
- 클리어 조건별 별점:
  - ★ 클리어
  - ★★ lives 50% 이상 유지
  - ★★★ lives 손실 0 (퍼펙트)
- 별점은 localStorage, 스테이지 맵에서 한눈에 표시
- 모든 스테이지 3★ 달성 시 "Master" 칭호 + 크리스탈 보너스
- 재도전 시 현재 별점 이하 달성하면 갱신, 이상이면 유지

#### A2. 7일 출석 보상
- 첫 런치 이후 매 24시간 = 출석 1일 카운트
- 7일 사이클 카드 (뽑기권/크리스탈/Prism 확률 부스터)
- 7일 완주 보상: **유니크 T4 타워** (메인 메뉴 배너로 자랑)
- 연속 출석 끊기면 day 1부터 (압박감)

#### A3. 위험 웨이브 (Danger Wave)
- 각 스테이지 마지막 웨이브를 "🚨 Danger Wave"로 표시
- 적 이동속도 ×1.15, 수 +2 (안 강해도 시각/사운드 변화 강조)
- BGM 템포 상승 또는 위험 스팅거 사운드
- 통과 시 "Waved Through!" 애니메이션 + 보너스 크리스탈

### 🥉 Tier B — 콘텐츠 확장 (3-5일)

#### B1. 프리스티지
- 크리스탈 1000 도달 시 "별똥별 포인트(SP)"로 환원 가능
- SP로 해금: 스킨(타워 이펙트), 부제(프로필), 영구 패시브
- 프리스티지 전환 시 메타 업그레이드 리셋 (감수 — 장기 플레이어용)

#### B2. 유니크 카드 히든 해금
- 도감에 ? 숨김 슬롯
- 조건 달성 시 해금:
  - 불사조: 화염 T3 조합 10회 (부활 특수)
  - 제로 카운터: 라이프 손실 0으로 캠페인 클리어
  - 로그 러너: Rush 5분 이내 클리어 10회
- 해금 시 화면 전체 애니메이션 "ACHIEVEMENT UNLOCKED"

#### B3. 주간 리더보드 (로컬 우선)
- 매주 월요일 00:00 리셋
- 이번 주 Rush 최단 시간/캠페인 최고 별점
- 시즌 종료 시 상위 10% 보상 (크리스탈 + 한정 스킨)
- 온라인 리더보드는 출시 후 서버 구축 (현재는 로컬 시뮬)

### 💎 Tier C — 깊은 콘텐츠 (1-2주)

#### C1. 세트 효과
- 같은 속성 3/5/7개 배치 시 자동 버프
  - Fire×3: 전역 burnDuration +20%
  - Electric×3: chainCount +1
  - Water×5: 슬로우 중첩 한도 +2
- UI: 맵 상단에 활성 세트 배지 표시

#### C2. 아티팩트/룬 시스템
- 메타 슬롯 5개, 각각 영구 패시브
- 크리스탈로 뽑기, 희귀도 4단계
- 예: "Neon Surge — 전투 시작 30초간 공격력 +20%"
- 장르 고인물 타겟. 출시 후 업데이트로 확장

#### C3. 간단한 스토리
- 각 스테이지 진입 시 5초 텍스트 장면
  - Stage 1: "네온 시티의 방어막이 붕괴됐다…"
  - Stage 3: "중앙 서버 침투 저지. 바이러스 감지."
  - Stage 6: "마지막 방어선. 당신이 유일한 희망."
- 클리어 후 크레딧 스타일 에필로그
- 세계관이 생기면 스킨/이벤트 명분 확보
