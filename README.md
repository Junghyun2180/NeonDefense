# 🌟 Neon Defense

네온 테마의 타워 디펜스 게임입니다. 가챠 시스템과 로그라이크 요소를 결합한 전략 게임입니다.

## 🎮 게임 방법

### 기본 조작
- **🎲 뽑기 (20G)**: 랜덤 Tier 1 타워 획득
- **⚡ 선택 조합**: 인벤토리에서 같은 타입 3개 선택 후 조합
- **🔄 전체 조합**: 조합 가능한 모든 타워 자동 조합
- **배치 (PC)**: 인벤토리에서 드래그하여 맵에 배치
- **배치 (모바일)**: 빈 타일 탭 → 속성 선택 → 티어 선택
- **🔮 타워 조합**: 맵에서 같은 타워 3개 선택 후 조합
- **💰 판매**: 맵의 타워 선택 후 판매 (50% 환급)

### 게임 진행
1. 타워를 뽑고 조합하여 강화
2. 맵에 타워를 배치
3. ▶ 시작 버튼으로 웨이브 시작
4. 5웨이브 클리어 시 다음 스테이지로 이동
5. 라이프가 0이 되면 게임 오버

## 💡 전략 팁

1. **디버프 적 우선 처치**: 📡 방해자와 🛡️ 억제자가 나타나면 빠르게 처치하세요
2. **슬로우 + 화상 조합**: ❄️ 냉기로 적을 늦추고 🔥 화염으로 지속 데미지!
3. **전격은 다수에게 효과적**: ⚡ 체인 라이트닝으로 여러 적을 한번에 공격
4. **경로 교차점에 배치**: 여러 경로가 만나는 곳에 타워를 배치하면 효율적
5. **높은 티어 타워 우선**: T4 타워 한 개가 T1 여러 개보다 강력합니다

## 📁 파일 구조

```
neon-defense/
├── index.html          # 메인 HTML (단일 파일 버전)
├── README.md           # 이 파일
├── css/
│   └── styles.css      # 스타일시트
└── js/
    ├── App.js          # 메인 게임 컴포넌트
    ├── components.js   # UI 컴포넌트
    ├── constants.js    # 게임 상수
    ├── drag-drop.js    # 드래그 앤 드롭 로직
    ├── enemy.js        # 적 관련 로직
    ├── game-engine.js  # 게임 루프
    ├── tower.js        # 타워 관련 로직
    └── utils.js        # 유틸리티 함수
```
