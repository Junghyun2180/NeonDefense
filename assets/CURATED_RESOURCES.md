## ⚠️ 주의사항

- **Kenney CC0** 외에는 라이선스 페이지 **직접 확인 필수** — 같은 사이트 내에서도 자산마다 다를 수 있음
- **CC-BY** 자산 사용 시 게임 내 **크레딧 화면**에 출처 표기 필수 (예: "Status icons by game-icons.net (CC BY 3.0)")
- **CC-BY-SA, GPL, CC-BY-NC** 라이선스 자산은 이 게임에 사용 금지 (SA 전염 / 비상업만)
- **itch.io 무료 ≠ 상업 OK** — 페이지 하단 "License" 섹션 확인
- 다운로드 시 **라이선스 페이지를 캡처/저장**해두기 (자산 페이지가 사라질 수 있음)

## 📥 다운로드 후 통합 가이드

이 게임의 자산 경로 규칙(`assets/SKIN_GUIDE.md` 참고):

```
다운로드한 PNG → 게임 경로

drone_1.png        → assets/enemies/normal.png
drone_fast.png     → assets/enemies/fast.png
drone_boss.png     → assets/enemies/boss.png
explosion_anim.png → assets/effects/explosion.png
fire_icon.png      → assets/icons/elements/fire.png
burn_status.png    → assets/icons/status/burn.png
```

- **권장 사이즈**: 타워 256×256 / 적 128×128 / 아이콘 32~64 (자세한 건 SKIN_GUIDE.md)
- **투명 배경 필수**: 타워·적·아이콘·이펙트 (타일·배경은 불투명 OK)
- **드래그 앤 드롭만으로 적용** — 코드 수정 불필요
