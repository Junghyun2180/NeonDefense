# 무료 2D 스프라이트 자산 사이트 조사 (상업 활용 가능)

Neon Defense의 스킨 교체 또는 새 자산 보충을 위한 외부 자산 소스 정리. 모두 **상업 사용 가능** 라이선스 보유 사이트만 선별.

---

## 🥇 Tier 1 — 즉시 사용 가능 (CC0 / 무제한 상업)

### Kenney.nl ⭐ 최우선 추천
- **URL**: https://kenney.nl/
- **자산 수**: 60,000+ (2D 스프라이트, 3D 모델, 사운드 등)
- **라이선스**: **CC0 (Public Domain)** — 출처 표기 불필요, 상업/수정/재배포 자유
- **장점**:
  - 모든 팩이 **일관된 스타일** → 여러 팩 혼합해 한 게임 구성 가능
  - **Tower Defense (Top-Down)** 전용 팩 존재
  - 가입 없이 바로 다운로드
- **단점**: 스타일이 정해져 있음 (대부분 만화풍/Lego 같은 깔끔한 도형)
- **이 게임 적합도**: ★★★★ (Tower Defense Top-Down 팩이 그대로 활용 가능)

### OpenGameArt.org
- **URL**: https://opengameart.org/
- **자산 수**: 40,000+
- **라이선스**: 자산마다 다름 (CC0, CC-BY, CC-BY-SA, GPL 등). **CC0 필터** 사용 권장.
- **장점**:
  - 픽셀 아트, 로우폴리, 컨셉 아트 등 다양한 스타일
  - 검색 시 라이선스 필터 가능
- **단점**:
  - 자산 품질 편차 큼 (인디 기여)
  - 검색 도구가 아쉬움 (운영자도 인정)
  - **라이선스 일일이 확인 필수** — CC-BY-SA는 게임 전체에 SA 적용 위험
- **이 게임 적합도**: ★★★ (CC0만 추려서 쓰면 좋음)

### itch.io (CC0 필터)
- **URL**: https://itch.io/game-assets/assets-cc0
- **자산 수**: 60,000+ (그 중 CC0 다수)
- **라이선스**: 자산마다 다름 — **반드시 페이지에서 확인**. CC0 필터로 좁히기.
- **장점**:
  - 트렌디한 픽셀/벡터/도트 아트 풍부
  - 네온/사이버펑크/신스웨이브 키워드로 검색 시 양질의 결과
  - "Neon" 태그: https://itch.io/game-assets/free/tag-neon
  - "Cyberpunk" 태그: https://itch.io/game-assets/free/tag-cyberpunk
  - Tower Defense 태그: https://itch.io/game-assets/free/tag-tower-defense
- **단점**:
  - 라이선스 명시 안 된 무료 자산도 있음 → 사용 불가
  - "Free" ≠ "Commercial OK". 페이지에서 라이선스 항목 확인 필수.
- **이 게임 적합도**: ★★★★★ (네온/사이버펑크 스타일 풍부)

## 🥈 Tier 2 — 무료 + 상업 OK (출처 표기 필요할 수 있음)

### CraftPix.net
- **URL**: https://craftpix.net/
- **무료 섹션**: https://craftpix.net/freebies/
- **라이선스**: **상업 사용 OK, 재배포 금지**. 출처 표기 불필요.
- **장점**:
  - 고품질 일관 스타일 (대부분 같은 작가팀)
  - **Tower Defense 전용** 무료 팩 다수 (`free-stone-tower-game-assets` 등)
  - 2D Top-Down 시점 자산 풍부
- **단점**: 무료 섹션은 제한적, 풀세트는 유료
- **이 게임 적합도**: ★★★★ (TD 전용 무료 팩 활용 가능)

### Game-Icons.net
- **URL**: https://game-icons.net/
- **자산 수**: 4,000+ 단색 SVG 아이콘
- **라이선스**: **CC-BY 3.0** (출처 표기 필요)
- **장점**:
  - 상태이상/스킬/타워 아이콘 매우 풍부 (이 게임의 status 아이콘 보완에 적합)
  - SVG라 색상 자유 변경 가능
- **단점**: 단색 윤곽선만 (네온 글로우는 직접 적용 필요)
- **이 게임 적합도**: ★★★★ (상태이상 12종 + 능력 아이콘에 즉시 활용)

## 🥉 Tier 3 — 보조 / 특수 용도

### LPC (Liberated Pixel Cup) Sprite Repository
- **URL**: https://lpc.opengameart.org/
- **라이선스**: CC-BY-SA 3.0 / GPL 3.0 (퍼지는 라이선스)
- **장점**: 캐릭터 애니메이션 풍부 (방향별 8 frame walk 등)
- **단점**: SA 라이선스라 게임 전체에 영향 가능 — **신중**
- **적합도**: ★★ (적 애니메이션 참고 정도)

### GDQuest GitHub 자산
- **URL**: https://github.com/gdquest-demos
- **라이선스**: **CC0**
- **장점**: Godot 친화 자산, 깔끔한 픽셀 아트
- **단점**: 패키지 단위가 적음
- **적합도**: ★★★

### awesome-cc0 리스트
- **URL**: https://github.com/madjin/awesome-cc0
- 다수의 CC0 자산 사이트 큐레이션 — 추가 발굴 출발점

### Freepik
- **URL**: https://www.freepik.com/
- **라이선스**: **무료 계정은 출처 표기 필수**, 유료는 표기 불요
- **단점**: 게임 자산보다는 일러스트레이션 중심, 스프라이트 적음
- **적합도**: ★★ (배경/UI 패널 정도)

## 🛡️ 라이선스 종류 빠른 참조

| 라이선스 | 상업 OK | 수정 OK | 출처 표기 | 비고 |
|---|---|---|---|---|
| **CC0** | ✅ | ✅ | 불필요 | 사실상 Public Domain — 가장 자유 |
| **CC-BY** | ✅ | ✅ | 필요 | 크레딧 표기만 하면 OK |
| **CC-BY-SA** | ✅ | ✅ | 필요 + **결과물도 동일 라이선스** | 게임 전체에 SA 전염 위험 |
| **CC-BY-NC** | ❌ | ✅ | 필요 | NC = NonCommercial. **상업 불가** |
| **GPL** | ✅ | ✅ | 필요 | 결과물도 GPL — 코드 공개 의무 가능 |
| **Asset 사이트 자체 라이선스** | 사이트마다 | 사이트마다 | — | EULA 직접 확인 |

⚠️ **CC-BY-NC 와 NoDerivs(ND)는 상업 게임에 절대 사용 금지**

## 📋 자산 도입 체크리스트

새 자산을 게임에 넣기 전 반드시 확인:

- [ ] 라이선스가 상업 사용 허용인가? (CC0 / CC-BY / 사이트 EULA 명시)
- [ ] 출처 표기 의무인 경우 게임 내 크레딧 페이지에 추가했나?
- [ ] CC-BY-SA / GPL 라이선스로 인해 게임 전체가 영향받는가? (대개 피하는 게 안전)
- [ ] 자산이 표시한 이름·스타일이 다른 상표와 충돌하지 않는가?
- [ ] AI 생성 자산인 경우 해당 사이트가 AI 자산 허용 여부?
- [ ] 다운로드 페이지의 라이선스 캡처/저장해둠 (사이트 자산이 사라질 수 있음)

## 🎯 추천 진행 순서

1. **Kenney "Tower Defense (Top-Down)"** 무료 팩 다운로드 → 스타일 비교용 레퍼런스
2. **itch.io "Cyberpunk + 2D + Sprites + CC0 필터"** 검색 → 네온 스타일 자산 발굴
3. **Game-Icons.net** 에서 상태이상 + 능력 아이콘 보강
4. **CraftPix.net 무료 TD 팩** 다운로드 → 부족한 보스/적 보충

이 4개 사이트만으로 90% 이상의 자산 수요 해결 가능합니다.
